# PhishGuard AI - AI & ML Engine

This directory contains the machine learning, deep learning, NLP, and computer vision models used for phishing detection.

## Purpose

To analyze suspicious URLs, emails, SMS content, and webpage screenshots using high-performance machine learning and deep learning models to predict phishing probability.

## Planned Contents

* **Model Architectures**: NLP models for email content analysis, Random Forest/XGBoost models for lexical URL features, and Computer Vision models for website visual spoofing/brand detection.
* **Inference Pipeline**: High-throughput processing scripts to analyze inputs and return predictive risk scores.
* **Training & Fine-tuning Scripts**: Code for training models on public/proprietary datasets and updating them with new threat intelligence.

## Planned Technologies

* **Language**: Python
* **Deep Learning Framework**: PyTorch
* **Natural Language Processing**: Hugging Face Transformers (BERT, RoBERTa, etc.)
* **Classical Machine Learning**: Scikit-learn & XGBoost
* **Computer Vision**: OpenCV / PyTorch-based CNNs or Vision Transformers

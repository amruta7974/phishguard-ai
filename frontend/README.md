# PhishGuard AI - Frontend UI Client

This directory contains the user interface and client-facing web application for the PhishGuard AI platform.

## Purpose

To provide a responsive, real-time, and modern dashboard for users, security analysts, and administrators to monitor, analyze, and manage phishing threats, system alerts, and threat intelligence.

## Planned Contents

* **Pages & Routing**: Dashboard, threat logs, domain analysis detail pages, settings, and authentication screens.
* **Components**: Reusable UI blocks, charts, data tables, and interactive visualization widgets.
* **State Management & Hooks**: Custom hooks for fetching real-time threat alerts, WebSockets, and system configuration.

## Planned Technologies

* **Framework**: Next.js (App Router)
* **Library**: React
* **Language**: TypeScript
* **Styling**: Tailwind CSS
* **UI Components**: shadcn/ui & Radix UI
* **Charts/Visuals**: Recharts
